Sistema completo de comercio electrónico especializado en hardware de computadoras

Este es mi primer proyecto desarrollado como parte de mi proceso de aprendizaje. Implementa un sistema de eCommerce completo con arquitectura en capas, patrones de diseño y buenas prácticas de desarrollo.

ASP.NET Core 8 MVC Entity Framework Core SQL Server Bootstrap 5 jQuery QuestPDF

Características Principales

Sistema de Ventas

Catálogo completo, carrito de compras, checkout y gestión de pedidos

Arma tu PC

Sistema interactivo paso a paso para poder armar una PC a medida del presupuesto y necesidad de cada cliente

Panel Admin
