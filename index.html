Mis Proyectos

🩺 Hospital San Juan - Sistema de Gestión de Turnos Médicos

API REST con .NET 9 + SPA React TypeScript | Clean Architecture | Repositorios separados (Backend/Frontend)

Sistema completo de gestión de turnos médicos que permite a pacientes reservar citas, a profesionales gestionar sus horarios y a administradores supervisar todo el sistema.

.NET 9.0 Web API React 19.2 TypeScript 5.9 SQL Server Tailwind CSS Docker

✨ Características Principales

🔐 Autenticación JWT

Sistema completo con roles (Admin, Professional, Patient), tokens y autorización

🏗️ Clean Architecture

Separación de responsabilidades: Domain, Application, Infrastructure y API

📅 Gestión de Turnos

Reserva, cancelación y seguimiento de turnos con validación de disponibilidad horaria

⚡ Frontend Moderno

SPA con React 19, TypeScript, React Query para cache inteligente y Zustand para estado global

📊 Panel Profesional

Los profesionales gestionan horarios semanales y actualizan estados de turnos

🎨 UI Responsive

Diseño mobile-first con Tailwind CSS, optimizado para todos los dispositivos

🎯 Qué aprendí y apliqué en este proyecto:

  • ✓ Clean Architecture con separación de Domain, Application, Infrastructure y Presentation
  • ✓ Diseño de API REST completa con DTOs y documentación Swagger/OpenAPI
  • ✓ Autenticación JWT con roles y autorización basada en políticas
  • ✓ Entity Framework Core con Code First, migraciones y relaciones complejas
  • ✓ React 19 con TypeScript para type-safety completo en frontend
  • ✓ Gestión de estado con React Query (servidor) y Zustand (global)
  • ✓ Containerización con Docker y CI/CD con GitHub Actions
